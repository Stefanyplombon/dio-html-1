Títulos e Textos

A tag <p> é usada para criar parágrafos. Ela organiza o texto em blocos separados.

Este é um exemplo de parágrafo criado com a tag <p>.

As tags <h1> até <h6> são usadas para definir títulos e subtítulos. O <h1> é o título principal e maior, enquanto o <h6> é o menor.

Tipos de Listas

As listas podem ser ordenadas (<ol>) ou não ordenadas (<ul>).

Os itens da lista são inseridos com a tag <li> dentro das listas.

Lista Ordenada:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista Não Ordenada:

  • Item A
  • Item B
  • Item C

Citação

A tag <blockquote> é usada para exibir uma citação longa, com recuo no texto.

"A educação é a arma mais poderosa que você pode usar para mudar o mundo." – Nelson Mandela

Links

A tag <a> cria um link. Pode ser usada para:

  • Link externo: leva para outro site
  • Link interno: leva para outra página do mesmo site
  • Âncora: leva para outra parte da mesma página
